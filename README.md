# Curriculum Vitae

Sebuah website Curriculum Vitae modern dengan efek parallax, animasi interaktif, navigasi melayang, serta kartu 3D tilt yang memberikan kesan profesional dan futuristik.  
Dibangun menggunakan **HTML, CSS, dan JavaScript murni** tanpa framework.

Website ini didesain sebagai CV online sekaligus portofolio personal untuk menampilkan profil, pendidikan, keahlian, dan proyek-proyek Bro Dika.

---

## Fitur Utama

### 1. **Parallax Background**
Latar belakang full-screen yang memberikan efek depth/ketebalan saat pengguna scroll.

### 2. **Navigasi Melayang (Sticky Navbar)**
- Navbar transparan dengan blur.
- Auto-shrink ketika discroll.
- Menu hamburger untuk mobile.

### 3. **Animasi Fade & Scroll Reveal**
Setiap section dan kartu muncul dengan animasi halus saat masuk viewport.

### 4. **3D Tilt Card Interaction**
Kartu portofolio responsif terhadap gerakan mouse, menciptakan efek 3D real-time.

### 5. **Fully Responsive**
Tampilan nyaman di smartphone, tablet, hingga desktop.

### 6. **Smooth Hero Section**
Dengan animasi:
- Fade slide pada nama
- Subjudul
- Deskripsi singkat

### 7. **Section Lengkap**
- Hero  
- Pendidikan  
- Keahlian  
- Portofolio (lengkap dengan link project)  
- Kontak
